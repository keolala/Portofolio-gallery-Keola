Smart Farming IoT

Built a basic web interface using ESP8266 to display temperature and soil data in real-time. This sparked my interest in frontend development and user interaction.Developed collaboratively during a Project-Based Learning (PJBL) program in junior high school, this project won The Best PJBL 1 award.

Main features:

  • DHT22 sensor for air temperature and humidity
  • Dallas Temperature sensor for soil temperature
  • Analog soil moisture sensor
  • Automatic water pump controlled via ESP8266 web server
  • WiFi Access Point and monitoring web

Technology Used: ESP8266, DHT22 sensor, Dallas Temperature sensor, HTML, CSS, JavaScript

Code Snippet:

// Setup WiFi Access Point
WiFi.softAP(ssid, password);
WiFi.softAPConfig(local_ip, gateway, subnet);
server.on("/", handle_OnConnect);
server.begin();

// Loop utama
void loop() {
  server.handleClient();
  sensors.requestTemperatures();
  temp = dht.readTemperature();
  hum = dht.readHumidity();
  kelembapanTanah = analogRead(pinSensorSoil);
  persentasekelembapanTanah = map(kelembapanTanah, 330, 740, 100, 1);

  if (Pumpotomatisstatus && persentasekelembapanTanah 60) {
    digitalWrite(pinPump, HIGH);
  } else {
    digitalWrite(pinPump, LOW);
  }

  lcd.setCursor(0, 0);
  lcd.print("Suhu: ");
  lcd.print(temp);
  lcd.print(" C");
}
